# 通用类总览 classes

[源码](https://github.com/ModyQyW/mp-scss/blob/main/classes)

## 使用

引入`classes`后即可使用内置的通用类。`classes`可以单独使用。

```scss
@import "~@modyqyw/mp-scss";
// @import "~@modyqyw/mp-scss/classes";
```

如果需要自定义，那么需要在引入之前写下你的自定义代码。

```scss
// 写下你的自定义代码

// 引入
@import "~@modyqyw/mp-scss";
// @import "~@modyqyw/mp-scss/classes";
```
